# Haiilo Checkout

A minimal, Haiilo-branded checkout kata built with Angular 20.

## Live Demo

[haiilo-checkoutapp-saif.netlify.app](https://haiilo-checkoutapp-saif.netlify.app/)

## Features

- Scan items and manage cart
- Dedicated service for managing the prices
- Real-time total calculation
- Responsive, Haiilo branded UI

## Tech Stack

- Angular 20
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm start
```

## Build

```bash
ng build
```

## Test

```bash
ng test
```
